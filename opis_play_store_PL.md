## ── POLSKA WERSJA (tasma-tracker) ────────────────────────────────────────────

### Nazwa aplikacji
Taśma Tracker

### Nazwa dewelopera
SkuDev

### Krótki opis (80 znaków max)
Tracker kuponów bukmacherskich z analizą EV, Kelly i statystykami

### Pełny opis (4000 znaków max)
Taśma Tracker to kompleksowe narzędzie dla graczy bukmacherskich, którzy chcą śledzić swoje kupony, analizować wyniki i podejmować lepsze decyzje oparte na danych.

🎯 GŁÓWNE FUNKCJE

📊 Śledzenie kuponów
• Szybkie dodawanie kuponów z kursem, stawką i zdarzeniami (legs)
• Obsługa freebet (SNR/SR), cashout i podatku od wygranych
• Historia zmian kursu przy edycji kuponu
• Szablony dla Twoich standardowych typów

📈 Analiza EV (Expected Value)
• Automatyczne obliczanie EV i przewagi (edge) dla każdego kuponu
• Badge VALUE / NEUTRAL / BAD przy każdym kuponie
• Własne prawdopodobieństwo lub automatyczne (implied probability)
• Podsumowanie EV w zakładce Analiza

📅 Kalendarz z heatmapą
• Miesięczny widok z kolorami dni (zielony/czerwony/żółty)
• Roczna heatmapa intensywności jak GitHub
• Kliknięcie dnia otwiera szczegóły i umożliwia rozliczenie kuponów

💹 Zaawansowane statystyki
• Wykres bankrolla z linią trendu 7-dniową
• Serie przegranych/wygranych z alertami
• Skuteczność po dniu tygodnia
• Porównanie miesięcy z delta ROI vs poprzedni miesiąc
• Zakres czasowy: ostatnie 7/30/90 dni lub wszystko
• Analiza długości taśmy vs skuteczność

🧮 Kalkulator Kelly
• Tryby: pełny / ½ / ¼ Kelly (domyślny bezpieczny)
• Segmenty: single, ako, zakresy kursów (<2.0 / 2.0–5.0 / >5.0)
• Ostrzeżenie przy małej próbce (< 20 zakładów)
• Twardy cap 5% bankrolla

🔍 Wyszukiwanie i filtrowanie
• Szukaj po dacie, buku, notatce
• Filtruj wg: bukmacher / status / EV

🚨 Inteligentne alerty
• Seria 5/10/20 przegranych z rekomendacjami
• Alert o nierozliczonych kuponach z poprzednich dni
• Dzienny limit straty + dzienny cel zysku
• Tygodniowy limit straty

🏆 Osiągnięcia
• 12 odznaka: pierwsza wygrana, serie, zyski, rekordy kursów

🎯 System jednostek
• Wyświetlanie stawek jako "Xu" zamiast wartości pieniężnej

📤 Eksport / Import
• Pełny backup JSON (wszystkie dane, ustawienia, szablony)
• Eksport CSV do Excela
• Eksport PDF z raportem miesięcznym
